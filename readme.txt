Requiriments:

Python
Pip (biblioteca)
Flask (biblioteca)

How to install:

Create virtual ambient: python -m venv [name]

Activate virtual ambient: 
PowerShell: C:\> <venv>\Scripts\Activate.ps1
CMD: <venv>\Scripts\activate.bat

How to installation Flask (required virtual ambient): pip install Flask
How to check installation Flask: flask --version

Create variable ambient: $env:FLASK_APP = "[appname].py"

Started App Flask: flask run

How to desactivate virtual ambient: deactivate
